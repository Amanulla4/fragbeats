# FragBeats 🎮🎵

> Game. Edit. Vibe.

A platform where gamers upload their best clips, add lo-fi beats, and share with a community that actually gets it.

## 🔗 Live Demo
[https://fragbeats.vercel.app](https://fragbeats.vercel.app)

## 📄 Pages
- 🏠 Homepage
- 🔐 Login / Signup
- 🔍 Explore — Search and filter clips
- 📤 Upload — 3 step clip upload flow
- 👤 Profile — Stats and clips
- 🎵 Music Library — 12 lo-fi tracks
- 💀 404 Page

## 🛠️ Built With
- React
- Vite
- Tailwind CSS
- React Router DOM
- Vercel (Deployment)

## ✨ Features
- Clean dark cyber UI
- Trending gaming clips feed
- Lo-fi music library
- Search and filter by game and genre
- 3 step upload flow
- User profile with stats
- Smooth scroll to top on navigation
- Custom 404 page
